Robotics Arm - Download

About the Project

This robotics arm was designed for precision and functionality. Controlled using Python and Arduino, it features:

  • Custom-built hardware with servo motors
  • Autonomous and remote-control functionality
  • Grip adjustment for different object sizes

Setup Instructions

  1. Download the project files using the button above.
  2. Follow the wiring diagram included in the project folder.
  3. Run the Python script to control the arm via your computer.
